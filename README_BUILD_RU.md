# Инструкция по сборке приложения для Windows

## Быстрая сборка

1. Дважды кликните на файл `build.bat`
   - Скрипт автоматически установит PyInstaller, зависимости и соберет исполняемый файл

2. Готовый файл находится в папке: `dist\PDF_Splitter.exe`

## Ручная сборка

Если предпочитаете собирать вручную:

```bash
# 1. Установить PyInstaller
pip install pyinstaller

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Собрать исполняемый файл
pyinstaller --onefile --windowed --name "PDF_Splitter" pdf_splitter_gui.py
```

## Продвинутая сборка (с использованием spec файла)

Для более точного контроля над процессом сборки:

```bash
pyinstaller build.spec
```

Или запустите `build_advanced.bat`

## Результат

Исполняемый файл будет создан в папке `dist`:
- `dist\PDF_Splitter.exe` - Один исполняемый файл (можно распространять отдельно)

## Важные замечания

- Исполняемый файл самодостаточен и не требует установки Python на целевой машине
- Первый запуск может занять несколько секунд, так как приложение извлекает временные файлы
- Windows Defender может пометить исполняемый файл при первом запуске - это нормально для исполняемых файлов PyInstaller. Можно добавить исключение при необходимости.

## Решение проблем

- Если сборка не удалась, убедитесь, что все зависимости установлены: `pip install -r requirements.txt`
- Если исполняемый файл не запускается, попробуйте собрать с консолью: замените `--windowed` на `--console` чтобы увидеть сообщения об ошибках
- При ложных срабатываниях антивируса может потребоваться подписать исполняемый файл (требуется сертификат для подписи кода)
